Case Study:

Building a Custom Website for Pursar ES

Pursar ES is a cloud-based platform that is being used primarily by financial institutions to manage their financial transactions such as money transfers, online payment, invoicing, etc. The platform is also used to manage the operations of financial institutions as well as serve the needs of their customers.

The founders of Pursar wanted to build an online presence with a website that would better showcase its services and provide a satisfactory user experience for visitors.

Challenge

The main challenge was to build a custom website that would meet the specific needs of Pursar ES and provide an attractive and professional look and feel. Additionally, the website needed to be easily maintainable and scalable, with the ability to add new pages and features as the company grows.

Role

  • - Frontend Developer
  • - Backend Developer
Responsibities & Features Implemented
  • I converted all mockup designs into functional web pages from scratch.
  • I implemented the UI of the pages using boostrap & CSS.
  • I created an admin dashboard to manage the website's content, with all the necessary backend functionalies for admin CRUD operations, and to create new blog posts.

Results & Technologies Used

  • - Bootstrap 4 / CSS
  • - AJAX
  • - Laravel
  • - JavaScript/jQuery
  • - MySQL
