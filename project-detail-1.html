Case Study:

Prototyping a Mobile and Web Interface for SAMZIO

SAMZIO is an imaginary company created for a group project for my design class. The idea was to come up with both mobile and web interfaces and a creative brief for the company. The company wanted to improve the mobile experience for its customers and provide a more convenient and user-friendly interface for shopping on the go. This interface was later implemented in HTML and CSS, following best practices.

Challenge

The main challenge was to prototype a mobile interface for SAMZIO that would provide an optimal shopping experience for customers, regardless of the device they were using. Additionally, the interface needed to be intuitive and easy to use, with a clear and streamlined navigation system. I also needed to come up with appropriate color palettes to represent the brand.

Role

  • - UI/UX Designer
  • - Frontend Developer
Responsibities & Features Implemented
  • I conducted user research to understand the needs and preferences of SAMZIO’S customers
  • Based on the user research, I designed a mobile and web interface that provided a clear and intuitive navigation system, with an appropriate color palette throughout the design.
  • I added interactivity to the prototype, such as animations and transitions, to provide a more engaging and realistic experience for users.
  • I conducted user testing to gather feedback on the prototype and to identify any areas for improvement.
  • I implemented the entire frontend interface.

Results & Technologies Used

  • - Adobe XD
  • - Adobe Illustrator
  • - HTML/CSS
